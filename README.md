**3.** Enter Your BOT-Token in **`Settings/config.json`**

### _Modify -> config.json_

```javascript
{
    "token": "Bot_Token",
}

```

**4.** after Fill BOT-Token in config, run **`setup.bat`**

**5.** start the bot with **`start.bat`**
